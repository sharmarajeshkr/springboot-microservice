Externalize Properties

Command line arg : Higher precedence (Space for Multiple arguments)
    EX : java -jar account-service-0.0.1-SNAPSHOT.jar --build.version="1.1"

JVM Properties (Lower than command Line)
    EX : java -Dbuild.version="1.1" -jar account-service-0.0.1-SNAPSHOT.jar

Environment Variables : Supported Universally
   In Windows OS
     env:BUILD_VERSION="1.1";java -jar account-service-0.0.1-SNAPSHOT.jar
   In Linux OS
     BUILD_VERSION="1.1";java -jar account-service-0.0.1-SNAPSHOT.jar
